# SHARP-shell – Cute Mini Shell by @Rioareos

Welcome to SHARP-shell, a colorful and playful command shell inspired and created by @Rioareos! This project is a cute-themed, feature-rich mini-shell made in Go for learning, self-expression, and fun command-line experiences.

## Features
- Cute ASCII banner and themed prompt
- Customizable color styles
- Built-in package and security utilities
- Maintenance & theme menu (`sharp --manage`)
- Fun mascot: ˃ 𖥦 ˂
- Easy to build and run

## Getting Started

1. **Install Go**: https://golang.org/dl/
2. **Clone repo:**
   ```sh
   git clone https://github.com/rioAreos/Sharpshell.git
   cd Sharpshell
   ```
3. **Build:**
   ```sh
   go build -o sharp-shell sharp.sh.go
   ```
4. **Run:**
   ```sh
   ./sharp-shell
   ```

## Example Prompt
```
SHARP ~/your/dir $ 
```
